DLC added: Nemesis, Aquatics Species Pack, Overlord, Toxoids, First Contact, Galactic Paragons, Astral Planes, The Machine Age, Cosmic Storms, Grand Archive.


Major changes:

Patch 3.14 implemented a limit on the number of habitats AI empires can construct. This was a critical change to improve late-game performance, as the excessive number of habitats in previous versions was a major hardware strain on the CPU.

Patch 3.14 introduced major engine optimizations to how ship modifiers and pop jobs are calculated. This reduced CPU wait times and improved late-game performance compared to earlier 3.x versions.

Storage required increased from around 10 GB to 12+ GB due to the addition of multiple DLCs and free updates.

Optimization of memory allocations within the alert manager and UI lowered RAM usage during heavy gameplay.


Why:

Stellaris developers optimized memory allocations within the alert manager between the 3.0 and 3.14 updates primarily to reduce CPU overhead, improve framerates, and mitigate late-game stuttering
